# Before Compile

ensure you have

- Ocaml
- GTK
- LablGTK

### MacosX install

1. brew install ocaml --with-x11
2. brew install opam
3. brew install gtk+
4. opam init
5. eval `opam config env`
6. opam switch 4.0.2.0
7. opem install lablgtk
8. then you can make program
